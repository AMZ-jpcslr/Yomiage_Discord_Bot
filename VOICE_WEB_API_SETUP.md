# VoiceVox Web API 設定ガイド

## 概要
VoiceVox Web APIを使用することで、ローカルサーバー不要で高速な音声読み上げが可能です。

## APIキーの取得

1. **APIキー取得サイトにアクセス**
   - https://su-shiki.com/api/

2. **APIキーを生成**
   - サイトの指示に従ってAPIキーを生成
   - 無料で利用可能（1日のポイント制限あり）

3. **環境変数に設定**
   ```
   VOICEVOX_API_KEY=your_api_key_here
   ```

## 使用方法

### 1. ボイスチャンネルに参加
```
/voice_web join
```

### 2. 音声設定の変更
```
/voice_web settings speaker:四国めたん（ノーマル） speed:0.9 intonation:1.1
```

### 3. APIステータス確認
```
/voice_web status
```

### 4. ボイスチャンネルから退出
```
/voice_web leave
```

## 主な特徴

- **高速処理**: クラウドベースで高速音声合成
- **サーバー不要**: ローカルVoiceVoxサーバーが不要
- **豊富な音声**: 四国めたん、ずんだもん等12種類の音声
- **細かい調整**: 速度、音程、イントネーションの調整可能

## ポイント消費

- 基本: 1500ポイント + 100ポイント × 文字数
- 例: 10文字のメッセージ = 1500 + (100 × 10) = 2500ポイント

## トラブルシューティング

### APIキーエラー
- 環境変数 `VOICEVOX_API_KEY` が正しく設定されているか確認
- APIキーが有効期限内か確認

### ポイント不足
- APIポイントの残量を確認
- 必要に応じて新しいAPIキーを取得

### 音声が再生されない
- ボットがボイスチャンネルに接続されているか確認
- FFmpegがインストールされているか確認
- 音声ファイルの権限を確認
