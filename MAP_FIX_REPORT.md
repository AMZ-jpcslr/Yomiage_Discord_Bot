# 地震マップ表示バグ修正報告

## 🐛 発見された問題

1. **参照マーカー（緑色の点）の不要な表示**
   - トカラ列島の参照点が地震に関係なく表示されていた

2. **震源地の重複表示**
   - 震源地に赤いXマークと震度円の両方が表示されていた
   - 震源地は赤いXマークのみであるべき

3. **観測点の震度が全て「2」で表示される**
   - 疑似震度データ生成で震源地にも震度を配置していた
   - 震度1以下を除外する処理で多くの地点が震度2になっていた

4. **ファイルパスエラー**
   - ビルド後の実行時に設定ファイルが見つからない

## ✅ 実施した修正

### 1. 参照マーカーの削除
```typescript
// 震源地位置の検証用マーカーを完全に削除
// トカラ列島の参照点表示コードを削除
```

### 2. 震源地の表示修正
```typescript
// 震源地付近の震度円描画処理を削除
// 震源地は赤いXマークのみで表示
```

### 3. 疑似震度データ生成の改善
```typescript
// 震源地自体には震度円を配置しない
const distances = [0.2, 0.5, 1.0, 1.5, 2.0, 3.0] // 0.1を除外
// 注意: 震源地には震度円を配置しない（赤いXマークのみ表示）
```

### 4. ファイルパス解決の修正
```typescript
// プロセスの作業ディレクトリを使用
const projectRoot = process.cwd()
const configPath = path.join(projectRoot, 'config/config.json')
const outputDir = path.join(process.cwd(), 'generated_images')
```
